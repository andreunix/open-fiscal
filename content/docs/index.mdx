## Estrutura de cada documento

Todo documento fiscal eletrônico segue a mesma lógica:

```
1. Emitente monta o XML conforme o leiaute (Anexo I do MOC)
2. Assina digitalmente com certificado ICP-Brasil
3. Transmite à SEFAZ autorizadora via Web Service (SOAP/HTTPS)
4. SEFAZ retorna protocolo de autorização (ou rejeição com código)
5. Emitente gera o documento auxiliar (DANFE / DACTE / DAMDFE)
```

## Sobre a Reforma Tributária

A partir de 2026, a Reforma Tributária (IBS, CBS, IS) impacta o leiaute de todos os documentos. A seção [Reforma Tributária](/docs/reforma) cobre as Notas Técnicas de transição.
