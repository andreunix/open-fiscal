Os conceitos que sustentam todo o resto: o que é o documento eletrônico, como se identifica e quais cadastros a SEFAZ consulta.

> **Em uma frase:** a nota fiscal é o XML autorizado, identificado por uma chave de 44 dígitos; o DANFE é só a representação auxiliar.
