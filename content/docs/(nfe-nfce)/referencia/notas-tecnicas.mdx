Esta página separa as Notas Técnicas em **duas listas**, usando como corte a consolidação do **MOC 7.0**:

- **MOC 7.0 — Visão Geral**, versão **7.00**, publicação **novembro de 2020** (atualizada com NTs publicadas até **outubro de 2020**);
- **MOC 7.0 — Anexo I**, capa **7.00 — novembro de 2020**, controle interno **7.03 — outubro de 2020**.

> **Critério:** uma NT é considerada **implementada (compilada)** quando a sua **versão vigente** já estava coberta pelo MOC 7.0 / Anexo I até o corte de **novembro de 2020**. Quando a revisão vigente é **posterior** ao corte, a NT entra como **overlay posterior ao MOC**, ainda que o número original seja antigo. A data exibida é a da **revisão vigente** (não a da 1ª publicação). Todos os overlays vigentes listados abaixo já foram incorporados às páginas temáticas do open-fiscal.

## Camadas para o open-fiscal

```txt
MOC 7.0 + Anexo I       = base até nov/2020 (65 NTs compiladas)
Overlays pós-MOC        = camada atual incorporada (36 NTs vigentes)
Schemas atuais          = fonte prática de validação XML
Tabelas vigentes        = fonte prática de códigos/domínios
```

> A data exibida é a da revisão vigente. Uma NT antiga (ex.: 2014.001, 2015.001) reaparece na lista de **overlays** porque sua revisão vigente é posterior a 2020. Confirme sempre a versão, os schemas e o cronograma de cada NT.
