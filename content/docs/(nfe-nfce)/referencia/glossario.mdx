## Campos de resposta

| Campo | O que representa |
|---|---|
| `cStat` | código do resultado do processamento |
| `xMotivo` | descrição textual associada ao resultado |
| `nProt` | número do protocolo, quando gerado |
| `nRec` | número do recibo de lote assíncrono |
| `tMed` | tempo médio indicado para nova consulta |

> Nunca decida apenas por `xMotivo`. Use `cStat` como chave e preserve ambos no diagnóstico.

## Ambientes e modelos

| Campo | Valor | Significado |
|---|---:|---|
| `tpAmb` | `1` | produção |
| `tpAmb` | `2` | homologação |
| `mod` | `55` | NF-e |
| `mod` | `65` | NFC-e |
| `tpEmis` | `1` | emissão normal |
| `tpEmis` | `4` | EPEC |
| `tpEmis` | `5` | FS-DA |
| `tpEmis` | `9` | off-line (NFC-e) |
