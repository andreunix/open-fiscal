> 🕒 **Conteúdo histórico.** O MOC 7.0 (2020) ilustra a EC 87/2015 com **partilha de 2016 (40% destino / 60% origem)**. Hoje a partilha do DIFAL é **100% destino**. Use este exemplo para entender a **mecânica e as fórmulas**, não os percentuais vigentes.

## O que o manual diz

Para **venda interestadual a consumidor final não contribuinte** (EC 87/2015, Convênio ICMS 93/2015, NT 2015.003):

| Sigla | Significado |
|---|---|
| BC | base de cálculo do ICMS |
| ALQ INTER | alíquota interestadual (4%, 7% ou 12%) |
| ALQ INTRA | alíquota interna na UF de destino |
| FCP | Fundo de Combate à Pobreza do destino |
| DIFAL | diferença entre a alíquota interna do destino e a interestadual |

### Fórmulas

```text
ICMS origem  = BC × ALQ INTER
DIFAL        = (BC × ALQ INTRA) − (BC × ALQ INTER)   (truncar o resultado)
vFCPUFDest   = BC × ALQ FCP
```

### Exemplo (alíquota interestadual de 7%, item 18% + FCP 2%)

```text
BC = 1.000,00 · ALQ INTER = 7% · ALQ INTRA = 18% · FCP = 2%
ICMS origem = 70,00
DIFAL = 180,00 − 70,00 = 110,00
vFCPUFDest = 20,00
Partilha 2016: destino 40% = 44,00 · origem 60% = 66,00   🕒
```

### Campos no XML

| Grupo | Campo | Valor do exemplo |
|---|---|---|
| `ICMSUFDest` | `vBCUFDest` | 1.000,00 |
| | `pICMSUFDest` | 18% |
| | `pICMSInter` | 7% |
| | `pICMSInterPart` | 40% 🕒 |
| | `pFCPUFDest` / `vFCPUFDest` | 2% / 20,00 |
| | `vICMSUFDest` (partilha destino) | 44,00 |
| | `vICMSUFRemet` (partilha origem) | 66,00 🕒 |
| `ICMSTot` | `vFCPUFDest`, `vICMSUFDest`, `vICMSUFRemet` | somatório dos itens |

## Como interpretar

A nota leva os valores **já partilhados** entre destino e remetente; o XML não "calcula", ele **declara** o resultado. O `truncar` no DIFAL é parte da regra — arredondar para cima diverge da SEFAZ.

## Vigência

- 🕒 `pICMSInterPart` e a partilha origem/destino do exemplo são de 2016; a transição da EC 87/2015 terminou (100% destino).
- 🔄 Alíquotas internas, interestaduais e FCP mudam por UF e por Informe Técnico — ver [Tabelas e códigos](/docs/referencia/tabelas-e-codigos).
- 🔄 A NT 2020.005 v1.21 **reativou** as regras `NA15-10` (valor do ICMS interestadual para a UF de destino, `vICMSUFDest`) e `NA17-10` (para a UF do remetente, `vICMSUFRemet`), conferindo o repasse do diferencial de alíquotas; ambas usam tolerância de R$ 0,01 e não se aplicam a NF-e de entrada (`tpNF=0`).
- 🔄 A NT 2022.005 v1.11 **reativou** a regra `NA01-20` (rejeição **694**), que volta a exigir o grupo `ICMSUFDest` em operação interestadual (`idDest=2`) a consumidor final (`indFinal=1`) não contribuinte (`indIEDest=9`); não se aplica ao Simples Nacional (`CRT=1`), a isentas/imunes/não tributadas, a devoluções que referenciem nota anterior a 2016, quando há partilha (`ICMSPart`), nem quando a UF de entrega (`entrega/UF`) é a do emitente. Pelo Ajuste SINIEF 18/2022, considere como destino a UF da **entrada física** (UF de entrega).

## Implicação de implementação

> **Implementação:** parametrize a partilha por **data de vigência** (não fixe 40/60). Trate alíquotas e FCP como tabelas versionadas por UF. Reproduza o **truncamento** do DIFAL exatamente como o manual.
